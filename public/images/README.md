# Images locales

Placez ici les images statiques de la plateforme.

## Images recommandées à télécharger

Pour héberger vos propres images (optionnel), placez-les dans ce dossier avec les noms suivants :

- `hero-books.jpg` — Photo d'allées de livres (page d'accueil)
- `librairie-interior.jpg` — Intérieur de librairie
- `books-stacked.jpg` — Livres empilés
- `reading-corner.jpg` — Coin lecture
- `library-shelves.jpg` — Rayonnages de bibliothèque
- `bookstore-client.jpg` — Client en librairie

## Utilisation

Les images placées ici sont accessibles via `/images/nom-du-fichier.jpg` dans le code.

Exemple :
```tsx
<Image src="/images/hero-books.jpg" alt="..." width={800} height={600} />
```
